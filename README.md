# ZX Labs Photography

个人摄影作品集网站。

## 项目结构

```
├── index.html      ← 访客网站（纯展示，不含管理代码）
├── admin.html      ← 作者管理后台（上传、删除、文件夹管理）
├── config.json     ← 网站配置（个人信息、分类、COS 地址等）
├── data.json       ← 照片数据（由作者后台自动更新）
├── CNAME           ← 自定义域名
└── README.md
```

## 文件职责

| 文件 | 改动频率 | 内容 |
|------|---------|------|
| config.json | 很少 | 站名、简介、联系方式、分类、COS/GitHub 配置、管理密码 |
| data.json | 每次发照片 | 照片列表（标题、分类、标签、手记等） |
| index.html | 基本不改 | 访客看到的展示页面 |
| admin.html | 基本不改 | 作者管理界面和上传逻辑 |

## 使用方法

1. 访问 `你的域名/admin.html` 或点网站右上角「作者」进入后台
2. 输入管理密码
3. 首次使用填写 GitHub Token 和腾讯云密钥
4. 选择/新建 COS 文件夹 → 选照片 → 填信息 → 发布
5. 发布逻辑：先上传图片到 COS，全部成功后才更新 data.json
